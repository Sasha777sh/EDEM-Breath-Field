## 🧬 这是什么

**EDEM** 是一个开放的科学技术项目，将物理学、生物学和意识结合在一起。

我们正在开发一种新物理学 — **"生命物理学"**，其中物质和意识是不可分割的。

**这不是哲学。这是一门新科学：**  
思想、呼吸和物质遵循统一的法则。

---

## ⚛️ 项目核心

我们已经证明生命遵循一个简单的公式：

```
生命 = 寂静 × (共振 × 交换) / 密度
```

**如果保持寂静**（内在存在），  
**共振增长**（与世界对齐）  
**密度降低**（阻力） —  
**一切都变得有生命。**
